# 노딱AI (NoddakAI) 🚗

> 주정차 위반 딱지 사진을 찍으면 AI가 이의신청서를 자동으로 작성해드립니다.

## 주요 기능

- 📸 **딱지 사진 분석** — 카메라 촬영 또는 갤러리 업로드
- 🤖 **AI 자동 인식** — 위반 유형, 일시, 장소, 과태료 자동 추출
- ⚖️ **법률 근거 제시** — 도로교통법 제32~35조, 제142조, 제160조 + 질서위반행위규제법 인용
- 📝 **이의신청서 생성** — 법적 근거 포함 이의신청서 초안 자동 작성
- 📋 **복사 & 저장** — 클립보드 복사 또는 HTML 파일 다운로드

## 기술 스택

- **Frontend**: Next.js 15, React 19, Tailwind CSS 4
- **AI**: Google Gemini 2.0 Flash (Vision)
- **Hosting**: Google Cloud Run (Seoul)
- **법률 DB**: 도로교통법 원문 내장 (시행 2026.1.1, 법률 제21016호)

## 법률 근거

프롬프트에 내장된 법률 원문:
- 도로교통법 제32조 (정차 및 주차의 금지)
- 도로교통법 제33조 (주차금지의 장소)
- 도로교통법 제34조~제34조의3 (방법, 시간, 특례, 경사지)
- 도로교통법 제35조 (주차위반에 대한 조치)
- 도로교통법 제142조 (행정소송과의 관계)
- 도로교통법 제160조 (과태료)
- 질서위반행위규제법 제16조, 제17조, 제20조, 제21조, 제24조의3
